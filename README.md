# Pedro Pathing Visualizer

Big Thanks to #16166 Watt's Up for developing this, we really appreciate your work.

Start ZIP files with 'npm run electron:dev'

# Most users can install with Curl/Homebrew:

Simply run:

curl -fsSL https://raw.githubusercontent.com/Mallen220/PedroPathingVisualizer/main/install.sh | bash

In a terminal and provide your password when prompted.

To update simply rerun the install script, it will automatically reinstall itself!

To uninstall navigate to the applications folder and delete "Pedro Pathing Visualizer.app"
